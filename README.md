# 🎵 ZenTune Pro
**Optimize your focus through emotional frequency.**

ZenTune Pro is a productivity dashboard that pairs your current mental state with curated YouTube audio streams. It features a unique "Flow Safeguard" to ensure your deep-work sessions are never interrupted by jarring alarms.

## ✨ Key Features
* **Emotional Calibration:** Choose from 9+ mental states (Focus, Anxiety Relief, Morning Kickstart, etc.).
* **Flow Safeguard:** Automatically transitions into "Overtime" mode when goals are reached to keep you in the zone.
* **Focus Mode:** A minimalist UI toggle that hides all distractions.
* **Smart Insights:** Tracks your total focus time, session counts, and dominant "Core Frequency."
* **Dark Mode:** Native support for late-night deep work.
